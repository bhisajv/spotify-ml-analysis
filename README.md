# Spotify Music Popularity Prediction

This project predicts whether a Spotify song is a hit or non-hit using machine learning.

Files included:
DM_PROJECT.ipynb – Code and analysis
DM_PROJECT.pdf – Research paper

Key results:
Decision Tree accuracy: 89.7%
Clustering silhouette score: 0.52

How to run:
Open the notebook file DM_PROJECT.ipynb in Jupyter Notebook.
